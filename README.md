# eClassProject
Στα πλαίσια της εργασίας υλοποιήθηκε ένας δυναµικός ιστοχώρος, µε στόχο την ενημέρωση των μαθητών για τα περιεχόμενα του μαθήματος και το εκπαιδευτικό υλικό, αλλά και την δυνατότητα επικοινωνίας με τους εκπαιδευτικούς. . Η ιστοσελίδα περιλαµβάνβει ένα αρχικό στάδιο σύνδεσης χρήστη, κατά το οποίο διαχωρίζονται οι λειτουργίες Student και Tutor. Στην συνέχεια υπάρχουν πέντε
σελίδες php που παρουσιάζουν την θεµατολογία της σελίδας, ενώ οι χρήστες ‘Tutor’ έχουν
την δυνατότητα να δηµιουργούν και να επεξεργάζονται περιεχόµενο, αλλά και νέους
χρήστες. Βασικός θεµατικός άξονας της σελίδας είναι η εκµάθηση JavaScript από εισαγωγικό επίπεδο, µε το εκπαιδευτικό υλικό να αποτελείται από σύνολα διαφανειών του Πανεπιστηµίου Πατρών και των καθηγητών Δ. Κουτσοµητρόπουλου και Ι. Γαροφαλάκη.

Η ιστοσελίδα βρίσκεται στον σύνδεσμο: https://kssotiriou.webpages.auth.gr/4263partB .
